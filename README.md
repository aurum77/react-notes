# react-notes

## What is this ?

- It's a Google Keep clone written using MongoDB, Express, React and Node

## How do I run this ?

1. Populate file `backend/.env` with `MONGODB_URI`
2. Run `npm run dev` or `yarn dev` at the folder `react-notes`
